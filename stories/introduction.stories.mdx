# Introduction to `linear-input`

Welcome to the `linear-input` library! This package is a highly customizable Web Component designed for input fields, built using the [Lit](https://lit.dev/) framework and following the best practices from [open-wc](https://open-wc.org/). Whether you're building a modern web application or adding custom elements to your project, `linear-input` offers flexibility and ease of use.

## Key Features

- **Customizable**: Easily style the input field to match your application's design.
- **Lightweight**: Built with the Lit framework, it ensures high performance and small bundle size.
- **Web Component**: Works natively in any web environment without dependencies on other frameworks.
- **TypeScript Support**: Fully typed

## Why `linear-input`?

The `linear-input` component is perfect for modern web applications that need flexible, reusable form elements. It is especially suitable for scenarios where customization and performance are critical. The component adheres to web standards, making it future-proof and highly interoperable.
